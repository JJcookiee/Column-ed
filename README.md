1) install xampp(you should already have this)
2) move project folder into xampp folder on local machine. The files path will look something similar to "C:\xampp\htdocs\Column-ed"
3) open xampp control panel
4) start mysql and apache
5) click mysql 'admin' button. this should open phpmyadmin
6) create new database and name it after the project
7) import schema.sql file into phpmyadmin. or just copy the code into the console
